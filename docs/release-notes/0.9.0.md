# PowerLens 0.9.0

PowerLens 0.9.0 is the first public preview of the macOS menu bar power
telemetry app.

## Highlights

- Live menu bar battery and power readout.
- Popover and dashboard views for battery state, power flow, diagnostics, and
  recent history.
- Power-flow visualization for adapter coverage, battery assist, charging,
  holding current, and battery-only discharge.
- Multiple menu bar display styles, including custom battery icons.
- Local-only telemetry history with no analytics or cloud service.
- Signed and notarized macOS release packaging support.

## Install

1. Download `PowerLens-0.9.0.dmg`.
2. Open the DMG.
3. Drag `PowerLens.app` into `Applications`.
4. Launch PowerLens from `Applications`.

## Privacy

PowerLens reads power and battery information locally. It does not send
analytics, telemetry, or crash data to a server. See `PRIVACY.md` for details.
